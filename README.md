# 🎓 Student Result Management System

A simple **Student Result Management System** built using **Python**. This console-based application allows users to add student records, view student details, check marks, and determine whether a student has passed or failed based on their marks.

## ✨ Features

- ➕ Add student names and marks
- 📋 View all student records
- 🔍 Search for a student's marks
- ✅ Check whether a student has **Passed** or **Failed**
- 💾 Stores student records in a text file (`students_list.txt`)
- 🖥️ Simple menu-driven interface

## 🛠️ Technologies Used

- Python 3
- File Handling

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Student-Result-Management.git
   ```
2. Open the project folder:
   ```bash
   cd Student-Result-Management
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## 📋 Menu Options

1. Add Student Details
2. View All Students
3. Check Student Marks
4. Check Pass/Fail Status
5. Exit

## 📌 Pass Criteria

- **Marks ≥ 40:** PASS ✅
- **Marks < 40:** FAIL ❌
